PAdES Electronic Signature Application

Overview

This application implements a qualified electronic signature system following the PAdES (PDF Advanced Electronic Signature) standard. It consists of two main components:

  1. Key Generator Application
  2. Signature Application

Features

  • RSA-4096 digital signatures
  • AES-256 encrypted private key storage
  • Automatic USB drive detection
  • PIN-protected key access
  • PDF document signing and verification
  • User-friendly GUI interface

System Architecture

Key Generator Application

The key generator application is responsible for:

  • Generating RSA-4096 key pairs
  • Encrypting private keys with AES-256
  • Storing encrypted private keys on USB drives
  • Storing public keys for verification

Signature Application

The main application provides:

  • Automatic USB drive detection
  • Private key loading and decryption
  • PDF document signing
  • Signature verification
  • Status monitoring and user feedback

Security Features

Cryptographic Implementation

  • RSA Keys: 4096-bit keys for maximum security
  • AES Encryption: 256-bit encryption for private key protection
  • Key Derivation: PBKDF2 with SHA-256 for PIN-based key generation
  • Hash Functions: SHA-256 for document hashing

USB Security

  • Automatic drive detection
  • Encrypted key storage
  • Secure key loading process
  • PIN protection

Components

1. Key Generator (key_generator/)

  • main.py: Application entry point
  • gui.py: User interface implementation
  • crypto.py: Cryptographic operations

2. Signature Application (signature_app/)

  • main.py: Application entry point
  • gui.py: Main application interface
  • crypto.py: Cryptographic operations
  • pdf_handler.py: PDF processing and signing
  • usb_detector.py: USB drive monitoring

Installation

Prerequisites

  • Python 3.x
  • PyQt5
  • cryptography
  • PyPDF2
  • psutil

Setup

# Install required packages
pip install -r requirements.txt
# Run key generator
python key_generator/main.py
# Run signature application
python signature_app/main.py

Usage Guide

Key Generation

  1. Launch key generator application
  2. Enter secure PIN (minimum 6 digits)
  3. Select USB drive for private key storage
  4. Select location for public key
  5. Generate key pair

Document Signing

  1. Launch signature application
  2. Insert USB drive with private key
  3. Enter PIN when prompted
  4. Select PDF to sign
  5. Choose output location
  6. Sign document

Signature Verification

  1. Launch signature application
  2. Select signed PDF
  3. Select corresponding public key
  4. Verify signature
